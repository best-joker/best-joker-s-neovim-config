## 快捷的操作
切换窗口：`(ctrl + w) + 方向键` ~~同时按 + 分开按~~
删除至行首、行尾：`ctrl + u` 、`ctrl + k`
立即应用：`:so`
开关文件树：`主键 + t`
新建文件：文件树窗口中下载`a`
开启终端：`主键 + \`    收起终端：`ctrl + \`
切换标签页（分割线）：`主键 + PgUp/PgDn`
关闭标签页：`主键 + w`
